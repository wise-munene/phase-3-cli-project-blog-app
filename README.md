# (Blog CLI App)

#### Date, 2025/12/4

#### By *Wise Munene*

## Description
Blog CLI is a command-line application that allows users to create accounts, write posts, comment on posts, and manage user data. It uses Python and SQLAlchemy for database operations, providing a simple interface for blog management without a web interface.
